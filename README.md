# jpyc_faucet_frontend

JPYC Faucetのフロントエンドです。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
.env_sampleファイルの中身を記入して

.env.localと.env.prodにリネームしてください


### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
